# Bookers2

## サイト概要
本の感想を投稿し、ユーザー同士で交流できるSNSアプリです。  
ユーザーは投稿に対して「いいね」「コメント」「評価」を行うことができ、  
フォロー機能やDM機能を通してユーザー同士でコミュニケーションができます。

---

## サイトテーマ
本のレビューを共有し、ユーザー同士が交流できるコミュニティサイト

---

## 主な機能

### ユーザー機能
- ユーザー登録 / ログイン
- プロフィール編集
- フォロー / フォロワー

### 投稿機能
- 本のレビュー投稿
- カテゴリタグ
- 閲覧数表示
- 5段階評価（Raty）
- いいね機能
- コメント機能

### コミュニケーション機能
- DM機能（相互フォローのみ）
- グループ作成
- グループ参加 / 退出
- グループメンバー表示

### 検索機能
- ユーザー検索
- 投稿検索
- カテゴリ検索

### ランキング
- 週間いいねランキング

### データ可視化
- ユーザー投稿数グラフ（Chart.js）
